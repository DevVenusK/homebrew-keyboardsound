# homebrew-keyboardsound

[KeyboardSound](https://github.com/DevVenusK/KeyboardSound) Homebrew 탭.

```bash
brew tap DevVenusK/keyboardsound
brew trust devvenusk/keyboardsound   # Homebrew 6.0+ 서드파티 탭 신뢰
brew install --cask keyboardsound
```

설치 후 시스템 설정 > 개인정보 보호 및 보안 > 입력 모니터링에서 KeyboardSound를 허용하세요.
이 빌드는 공증되지 않아 첫 실행 시 Gatekeeper 경고가 뜰 수 있습니다(앱 우클릭 → 열기).
